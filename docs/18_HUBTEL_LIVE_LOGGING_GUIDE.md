# Hubtel Live Logging Guide

This project now writes structured Hubtel integration logs to:

- `data/hubtel-live.log`

Format:

```text
[2026-03-02T11:42:32.311Z] [EVENT_TAG] { ...json payload... }
```

## Enable / Configure

Set in `.env`:

- `HUBTEL_LIVE_LOG_ENABLED=true`
- `HUBTEL_LIVE_LOG_FILE=./data/hubtel-live.log`
- `HUBTEL_LIVE_LOG_REDACT_SENSITIVE=true`
- `HUBTEL_LIVE_LOG_MASK_PHONE=false`

## Quick Commands

- Clear log: `npm run logs:hubtel:clear`
- Tail live: `npm run logs:hubtel:tail`
- Show latest: `npm run logs:hubtel:last`
